# Sprint Button

The **Sprint** button launches the `vailasprint.py` module, designed for high-performance biomechanical analysis of sprint running.

## Overview

This tool analyzes the kinematics and spatiotemporal parameters of sprint running, focusing on acceleration and maximal speed phases. It is used to extract performance metrics for athletes and researchers.

## Key Features

- **Kinematic Analysis:** Extracts joint angles and segment inclinations during the sprint cycle.
- **Spatiotemporal Metrics:** Calculates step length, step frequency, contact time, and flight time.
- **Phase Detection:** Automatically segments the sprint into acceleration and constant speed phases.
- **Visualization:** Generates plots of velocity-time and kinematic profiles.

## Usage

1. Click **Sprint** in Frame B.
2. Select the video or CSV data containing the sprint performance.
3. Configure the analysis parameters (sample rate, start/end frames).
4. Run the analysis to generate the report and processed data.
