Description

Deadlift/RDL biomechanical analysis for MediaPipe Pose CSV files. The tool exports processed frame metrics, diagnostic plots, variant classification, and an HTML report.

The wrist landmark is used as a barbell proxy because MediaPipe Pose does not track the bar directly. Use a sagittal/profile camera view for horizontal setup checks.

New validations

  • Arm verticality: shoulder-to-wrist horizontal offset at setup; tolerance 5 cm.
  • Bar over midfoot: wrist projection against heel-to-foot-index midpoint; tolerance 3 cm.
  • Initial pull synchronism: first 15% of concentric pull; flags knee extension velocity greater than 2x hip opening velocity.

Metrics

  • Stance width ratio
  • Knee flexion
  • Tibia angle to ground
  • Spine deviation from setup
  • Bar path proximity
  • Cervical deviation
  • Deadlift variant classification

Usage

GUI: run uv run vaila.py, click Deadlift, and select a folder containing MediaPipe CSV files.

uv run python vaila/vaila_deadlift.py -i path/to/pose.csv -o path/to/output

Outputs

  • Processed kinematics CSV
  • Lower limb and safety metric plots
  • Biomechanical HTML report
