How to Use

1. Run the Script

python vaila/viewc3d.py

2. Select C3D File

  • A dialog window will open to select the C3D file
  • The script will automatically detect units (mm/m)
  • You can confirm or change the unit conversion

3. Select Markers

  • Choose which markers you want to visualize
  • You can select all or specific markers only
  • Markers will be displayed as colored spheres

Keyboard Controls

Basic Controls

L - Set view limits (with adaptive ground plane)
R - Reset view
H - Show help
Q - Quit

Advanced Controls

F - Load custom field lines (CSV)
Mouse - Rotate, zoom and pan in the visualization

Advanced Features

Custom Field Lines

You can create CSV files with custom field lines. The format should include:
line_id,x1,y1,z1,x2,y2,z2,color_r,color_g,color_b,description 1,-50,0,0,50,0,0,1,0,0,center_line_red 2,0,-30,0,0,30,0,0,1,0,center_line_green
Note: Coordinates must be in the same unit system as the C3D data.

Enhanced Coordinate System

  • Colored axes (Red=X, Green=Y, Blue=Z)
  • Spheres at axis endpoints
  • Adaptive scale based on data

Adaptive Ground Plane

  • Always positioned at Z=0
  • Dimensions adjusted to data boundaries
  • Adaptive grid spacing

Inspirations and Improvements

This module incorporates visualization concepts from Blender and Mokka BTK:
  • Blender: Adaptive grid, flexible line drawing, robust coordinate system
  • Mokka BTK: Enhanced coordinate system, annotations, biomechanical visualization

Troubleshooting

Problem: Visualization doesn't appear
Solution: Check if Open3D is installed: pip install open3d
Problem: Ground plane in incorrect position
Solution: Use 'L' key to reset view limits
Problem: CSV file doesn't load
Solution: Check CSV format and if coordinates are correct
