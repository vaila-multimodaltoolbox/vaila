Markerless 2D YOLOv26 Pose Help

This tool provides 2D pose estimation using YOLOv26/v11 models. It is a fast and reliable alternative to MediaPipe for standard joint tracking.

Quick Start

  • Model: Select a YOLO pose model. yolo11x-pose.pt offers the highest accuracy.
  • Confidence: Set the minimum detection threshold (default 0.5). Lower it for distant subjects.
  • Directory: Choose the folder containing your videos. All videos inside will be processed.

Outputs

Results are saved in a folder named processed_yolo26_YYYYMMDD_HHMMSS:

  • Annotated Video: Visual verification of tracking quality with skeleton overlay.
  • Normalized CSV: Landmark coordinates from 0.0 to 1.0.
  • Pixel CSV: Landmark coordinates in exact pixel locations.
  • Log Info: Processing metadata and performance stats.
