SAM 3 not installed

Install the optional stack, then restart vailá.

Standard (CPU pyproject template)

uv sync --extra sam

Workstation (NVIDIA CUDA template)

After switching to the Linux/Windows CUDA pyproject (see install scripts or bin/use_pyproject_*):

uv sync --extra gpu --extra sam
Runtime: SAM 3 video inference needs an NVIDIA GPU with CUDA, even if the sam extra is installed.
Docs: Hybrid CPU laptop vs NVIDIA workstation — see AGENTS.md in the repository root.

Weights (Hugging Face, gated)

Accept the model on Hugging Face, then authenticate and download, for example:

uv run hf auth login
uv run vaila/vaila_sam.py --download-weights

Or: uv run hf download facebook/sam3 sam3.pt --local-dir vaila/models/sam3

FIFA Skeletal pipeline (optional)

For the FIFA Skeletal Tracking Light tools (vaila_sam.py fifa …):

uv sync --extra fifa
# With CUDA template:
uv sync --extra gpu --extra fifa

SAM 3D Body weights: vaila/models/sam-3d-dinov3/instruction_to_download_models.txt
