vailá — Tennis court visualization

Module: vaila/tennis_court.py

ITF-standard tennis court drawing with trajectory overlays and KDE heatmaps. Default reference CSV: vaila/models/tenniscourt_ref3d.csv (labels only; geometry is metric-accurate in code).

Court dimensions (ITF)

  • Length: 23.77 m
  • Doubles width: 10.97 m
  • Singles width: 8.23 m (1.37 m alley each side)
  • Service box length: 6.40 m (net to service line)

Manual markers

  • Left-click: place marker
  • Shift + left-click: next marker number
  • Right-click: delete nearest
  • Ctrl+S: save CSV

CLI

uv run python -m vaila.tennis_court --court <path> --markers <path> --heatmap
