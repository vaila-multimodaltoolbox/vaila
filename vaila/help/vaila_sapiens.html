vailá - Sapiens2 Pose (vaila_sapiens.py)

Version: 0.3.75  |  Updated: 2026-07-07

GUI: Frame B → YOLO + FB → Sapiens2 Pose

Top-down 308-keypoint human pose on video (Meta Sapiens2 + DETR detector).

License: Sapiens2 uses Meta's Sapiens2 License (not AGPL). Accept HF terms before download.

Setup

uv sync --extra sapiens
bash bin/setup_sapiens2.sh

Models & weight download

ModelNotes
1bDefault — fits RTX 4090 24 GiB
0.4b / 0.8bFaster, less VRAM
5bMax quality (~24 GiB checkpoint); heavy VRAM with many persons
# Default 1B
uv run vaila/vaila_sapiens.py --download-weights --model 1b

# Largest 5B (accept Meta license on Hugging Face first)
uv run vaila/vaila_sapiens.py --download-weights --model 5b

uv run vaila/vaila_sapiens.py -i video.mp4 -o out/ --model 5b --stride 1

GUI → CLI mirror

On Run, the terminal prints >> vaila/vaila_sapiens: Equivalent CLI with a copy-paste command.

Terminal progress (v0.3.74)

CLI runs print >> vaila/vaila_sapiens: status lines (model load, video geometry, CSV write) plus a tqdm frame bar on stderr. The first 1–2 minutes are often model load before the bar moves. Use --quiet to suppress the bar (GUI batch does this automatically and uses frame heartbeat lines instead).

getpixelvideo (v0.3.75)

  • Load wide pose: <stem>_id_NN_sapiens_pose.csv or <stem>_getpixelvideo_pose.csv
  • Load Tracking CSV long pose: <stem>_sapiens_vaila.csv (pick person_id)
  • Load Tracking CSV bboxes: sapiens_bbox_tracks.csv (SAM-compatible)
  • REC2D/REC3D: <stem>_markers.csv (foot anchor, one point per person)
