üìñ Description

Advanced 3D viewer for C3D files with adaptive visualization (Open3D backend). To run: uv run viewc3d.py. The toolbox also provides a PyVista-based viewer: viewc3d_pyvista.py (see help in that module).

Architecture

  • VailaModel (data layer): GetPointFrame, GetFrameNumber, GetPointFrequency, get_bounds(), is_z_up()
  • VailaView (Open3D layer): Scene, trails, dynamic grid, Z-up camera, marker picking

Key Features

  • Adaptive scale (lab to soccer field), automatic unit detection (mm/m)
  • Trails with ghosting (older segments darker), Z-up camera when Z range > Y range
  • Marker picking: ` / ~ to cycle and highlight one marker (yellow)
  • Grid and ground from data bounds; Matplotlib fallback if no OpenGL
