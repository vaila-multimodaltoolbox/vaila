This module provides native file dialogs that work without conflicts with Pygame.

    title="Select File",
    file_types=[("*.csv", "CSV Files"), ("*.txt", "Text Files")]
)

# Open yes/no dialog

    title="Question",
    message="Do you want to continue?"
