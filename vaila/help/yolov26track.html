üìñ Description

Object detection and tracking using YOLO v26 (latest version) with integrated pose estimation.

Features:

  • Object detection and tracking using Ultralytics YOLO26
  • Graphical interface (Tkinter) for configuration
  • Video processing with OpenCV
  • CSV generation with frame-by-frame tracking data
  • Pose estimation within tracked bounding boxes
  • Automatic GPU detection (CUDA/MPS/CPU)
