Description

brainstorm.py is a Tkinter workspace for audio transcription, prompt editing, music-code generation, image prompt generation, creative ideas, text-to-audio, and PDF transcription.

Main Actions

  • Record Audio — capture voice audio.
  • Transcribe and Batch Transcribe — convert audio to text.
  • Transcribe PDFs — open transcribe_pdfs.py for scanned or handwritten PDF exams.
  • Generate Music Code, Generate Image, and Creative Ideas — use the edited text as prompt material.

PDF Transcription Button

The separate PDF Transcription section near the top of Brainstorm opens the PDF transcription dialog with defaults for the PAE Biomec1 workflow when that folder exists.

Requirements

  • Audio workflows: tkinter, speech_recognition, sounddevice, soundfile, optional FFmpeg and MIDI/TTS packages.
  • PDF workflows: Poppler tools and Gemini CLI for vision transcription.
