DLT3D e Reconstrução 3D - vailá

Visão Geral: Este guia explica como usar os modules de calibração DLT3D e reconstrução 3D do vailá para analysis de movimento tridimensional.

Módulos Disponíveis

1. dlt3d.py - Calibração DLT3D

Função: Calcula os parâmetros DLT3D (11 parâmetros) para transformação de coordenadas de pixel para coordenadas 3D reais.

Como usar:

Passo 1: Execute o script dlt3d.py
Passo 2: Selecione o file de coordenadas de pixel (CSV)
Passo 3: Escolha se deseja criar um template REF3D ou usar um file existente
Passo 4: Se criar template, edite com as coordenadas 3D reais e execute novamente

Formatos de File:

Arquivo de Pixel:
frame,p1_x,p1_y,p2_x,p2_y,...,pN_x,pN_y
Arquivo REF3D:
frame,p1_x,p1_y,p1_z,p2_x,p2_y,p2_z,...,pN_x,pN_y,pN_z
Arquivo de Saída DLT3D:
frame,L1,L2,L3,L4,L5,L6,L7,L8,L9,L10,L11
Importante: O file REF3D deve conter coordenadas 3D reais (x, y, z) para cada ponto de calibração.

2. rec3d_one_dlt3d.py - Reconstrução 3D (Uma Linha)

Função: Reconstrução 3D usando um conjunto de parâmetros DLT3D por câmera (processamento de line única).

Características:

  • Usa um conjunto de parâmetros DLT3D por câmera
  • Processa múltiplas câmeras simultaneamente
  • Gera files CSV, 3D e C3D
  • Otimizado para grandes conjuntos de dados

Como usar:

Passo 1: Selecione os files DLT3D (um por câmera)
Passo 2: Selecione os files de coordenadas de pixel (um por câmera)
Passo 3: Escolha o diretório de saída
Passo 4: Defina a frequência de dados (Hz)
Saída: Arquivos CSV, 3D, C3D (metros) e C3D (milímetros)

3. rec3d.py - Reconstrução 3D (Processamento em Lote)

Função: Reconstrução 3D em lote usando múltiplos conjuntos de parâmetros DLT3D por câmera.

Características:

  • Processa múltiplos files CSV em um diretório
  • Suporta parâmetros DLT3D que variam por frame
  • Múltiplas câmeras com diferentes parâmetros por frame
  • Otimizado para processamento em lote

Como usar:

Passo 1: Selecione os files DLT3D (um por câmera)
Passo 2: Selecione o diretório com files CSV para processar
Passo 3: Escolha o diretório de saída
Passo 4: Defina a frequência de dados (Hz)
Saída: Arquivos .3d com coordenadas 3D reconstruídas

Diferenças entre os Módulos

rec3d_one_dlt3d.py: - Um conjunto de parâmetros DLT3D por câmera - Processa files individuais - Gera múltiplos formatos de saída (CSV, 3D, C3D) rec3d.py: - Múltiplos conjuntos de parâmetros DLT3D por câmera - Processamento em lote de múltiplos files - Parâmetros podem variar por frame - Saída em formato .3d

Fluxo de Trabalho Recomendado

1. Calibração: Use dlt3d.py para calcular os parâmetros DLT3D
2. Reconstrução Individual: Use rec3d_one_dlt3d.py para files únicos
3. Reconstrução em Lote: Use rec3d.py para múltiplos files

Formatos de Arquivo Suportados

Entrada:

  • CSV: Coordenadas de pixel (frame, p1_x, p1_y, p2_x, p2_y, ...)
  • DLT3D: Parâmetros DLT3D (frame, L1, L2, ..., L11)
  • REF3D: Coordenadas 3D de referência (frame, p1_x, p1_y, p1_z, ...)

Saída:

  • CSV: Coordenadas 3D reconstruídas
  • 3D: Formato específico do vailá
  • C3D: Formato padrão da indústria (metros e milímetros)
Dicas Importantes:
  • Certifique-se de que os files de pixel tenham o cabeçalho padrão do vailá
  • Verifique se as coordenadas 3D de referência estão corretas
  • Use a mesma frequência de dados em todo o processo
  • Mantenha backup dos files originais
