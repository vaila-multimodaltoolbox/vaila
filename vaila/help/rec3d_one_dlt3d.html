Description

Batch 3D reconstruction using DLT with multiple cameras. Each camera: one DLT3D parameter file and one pixel coordinate CSV. Output: CSV, .3d, and C3D (meters and mm).

GUI mode

Run with no arguments or --gui. You are asked for number of cameras, then one file dialog per camera for DLT3D files and one per camera for pixel CSVs, so files can be in different directories. Then choose output directory and data rate (Hz).

CLI mode

Use --dlt3d, --pixels, --output, and optionally --fps. File order must match (first DLT3D with first pixel CSV).

ArgumentDescription
--dlt3dDLT3D files (one per camera)
--pixelsPixel CSV files (one per camera)
--fpsPoint data rate Hz (default 100)
-o, --outputOutput directory
--guiLaunch GUI

Example:

python -m vaila.rec3d_one_dlt3d --dlt3d cam1.dlt3d cam2.dlt3d --pixels cam1.csv cam2.csv --fps 60 --output ./out
