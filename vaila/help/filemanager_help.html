Normalize Names (Rename)

This powerful feature automatically standardizes file and folder names in a selected directory. Warning: This action is irreversible!

What it does:

  • Lowercase: Converts all filenames and extensions to lowercase (e.g., VIDEO.MP4 ‚Üí video.mp4).
  • Remove Accents: Removes accents from characters (e.g., a√ß√£o.txt ‚Üí acao.txt).
  • Remove Dots: Removes dots from the filename itself to avoid confusion, keeping only the final dot for the extension (e.g., my.file.name.csv ‚Üí myfilename.csv).
  • Spaces to Underscores: Replaces spaces and hyphens with underscores (e.g., my file-name.txt ‚Üí my_file_name.txt).
  • Cleanup: Removes special characters and duplicate underscores.

Duplicate Safety:

If renaming results in a duplicate filename (e.g., file.txt and FILE.TXT resolving to the same name), vail√° will automatically append a counter (e.g., file_1.txt) to prevent overwriting or errors.

Copy / Move Files

Easily organize your data by copying or moving files based on extensions and name patterns.

  • Source Directory: Select the folder containing your files.
  • Extension: Specify the file type (e.g., .c3d, .csv). Leave blank for all files.
  • Patterns: Enter keywords (one per line) to filter files. Only files containing these patterns will be processed.

Files are organized into timestamped folders in the destination, preserving the original structure.

Remove Files

Safety Check: vail√° prevents the deletion of critical system files and directories (e.g., Windows, System32, /etc, .git).

Remove files or directories based on:

  • Extension: Deletes all files of a specific type (e.g., .tmp).
  • Directory Name: Removes entire folders matching a name pattern.
  • Filename Pattern: Removes specific files matching a wildcard pattern (e.g., *backup*).

Other Tools

  • Tree File: Generates a text file listing all files in a directory (filtered by extension), preserving the folder structure. Useful for cataloging datasets.
  • Find File: Search for files matching a specific pattern. (Currently in development/CLI mode).
  • Import/Export: A menu for converting data formats (e.g., TRC export, C3D import).
