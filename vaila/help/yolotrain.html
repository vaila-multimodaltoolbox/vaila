Usage (updated)

GUI: in the main window, open Frame B → YOLO and SAM → Train YOLO.

CLI (open GUI):

uv run python -m vaila.yolotrain

Ultralytics CLI (train / val / test / predict)

If you want fully headless training/evaluation, use Ultralytics directly:

# Detect train
uv run yolo detect train model=yolo11n.pt data=/ABS/path/to/data.yaml epochs=100 imgsz=640 device=0

# Validate (val split)
uv run yolo detect val model=/ABS/path/to/weights/best.pt data=/ABS/path/to/data.yaml split=val

# Evaluate (test split, if available)
uv run yolo detect val model=/ABS/path/to/weights/best.pt data=/ABS/path/to/data.yaml split=test

# Segmentation (mask) train
uv run yolo segment train model=yolo11n-seg.pt data=/ABS/path/to/data.yaml epochs=100 imgsz=640 device=0

# Pose train (keypoints)
uv run yolo pose train model=yolo26s-pose.pt data=/ABS/path/to/data.yaml epochs=200 imgsz=1280 device=0

# Predict on a video
uv run yolo predict model=/ABS/path/to/weights/best.pt source=/ABS/path/to/video.mp4 conf=0.25 device=0
