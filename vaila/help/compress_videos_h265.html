üìñ Description

Compresses videos in a specified directory to H.265 (HEVC) format using FFmpeg.

  • GPU acceleration with NVIDIA NVENC (hevc_nvenc, auto-detected)
  • macOS VideoToolbox support (hevc_videotoolbox)
  • CPU fallback to libx265 when no GPU is available
  • Better compression than H.264 ‚Äî up to 50% smaller files
  • Resolution control: keep original or downscale
  • Cross-platform: Windows, Linux, macOS

üöÄ CLI Usage
