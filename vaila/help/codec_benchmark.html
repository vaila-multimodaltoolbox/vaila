📖 Description

Systematically benchmarks H.264, H.265 (HEVC), and H.266 (VVC) codecs on a dataset of videos.

  • Automated comparison: Runs multiple codecs sequentially.
  • Performance metrics: Time, size, ratio.
  • Results storage: Saves to results.json.
  • Parallel execution: Process multiple videos at once.

🚀 CLI Usage

# Basic usage
python -m vaila.codec_benchmark --dir /path/to/videos

# High performance mode
python -m vaila.codec_benchmark --dir /path/to/videos --workers 4 --output experiment_1
OptionDefaultDescription
--dir(required)Source video directory
--workers1Parallel tasks
--outputbenchmark_resultsOutput directory
