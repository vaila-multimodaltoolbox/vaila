SAM 3D Body (facebook/sam-3d-body-dinov3) — accept the license on the Hugging Face model card.

1. https://huggingface.co/facebook/sam-3d-body-dinov3
2. uv run hf auth login
3. From the repository root:
   uv run hf download facebook/sam-3d-body-dinov3 --local-dir vaila/models/sam-3d-dinov3

This restores model.ckpt, assets/mhr_model.pt, and other files expected by the upstream project.
See also: CONTRIBUTING.md (vailá models directory) and https://github.com/facebookresearch/sam-3d-body
